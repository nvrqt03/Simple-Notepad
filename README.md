# Simple-Notepad

## This is a simple working notepad or todo list using the Room/SQLite database. 

![simple notepad](https://user-images.githubusercontent.com/22177989/81431134-458ee980-9126-11ea-9919-e77b038430f1.png)

#### Click on the action button to add your note.

![simple notepad add note](https://user-images.githubusercontent.com/22177989/81431149-49bb0700-9126-11ea-98b8-a0369a60ad34.png)

#### You can set the priority - how important is this note on a scale from 1 - 10. (10 is most important). Items will be ordered highest priortiy on top. 

![priority](https://user-images.githubusercontent.com/22177989/81431929-9a7f2f80-9127-11ea-82b3-aa4505f1413a.png)

#### Once you're done, click save at the top. 

![save](https://user-images.githubusercontent.com/22177989/81431918-97843f00-9127-11ea-998c-d8fd7b7aa2f4.png)

#### To delete a note, swipe left or right. To edit, click on the note to bring up the edit screen. X out or hit the back button if you don't want to save changes, or click save at the top to save changes. 

![edit](https://user-images.githubusercontent.com/22177989/81432184-18433b00-9128-11ea-97d8-b8010f3bbf2e.png)
